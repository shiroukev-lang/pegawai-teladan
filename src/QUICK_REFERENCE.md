## 🎯 Common Tasks

### Super Admin Tasks:
1. **Add New Employee:**
   - Go to "Pegawai" tab
   - Fill NIP, Nama, Bagian
   - Click "Tambah & Generate Link"
   - Copy the voting link

2. **Add Candidate:**
   - Go to "Kandidat" tab
   - Fill candidate details
   - Add photo URL (optional)
   - Click "Tambah Kandidat"

3. **Change Voting Phase:**
   - Go to "Settings" tab
   - Select Phase 1 or 2
   - Click "Ubah Phase"

4. **Reset Votes:**
   - Go to "Settings" tab
   - Choose reset option
   - Confirm action

### Admin Tasks:
1. **View Statistics:**
   - Go to "Overview" tab
   - See total employees & candidates
   - Check voting progress

2. **Check Results:**
   - Go to "Hasil Tahap 1" or "Hasil Tahap 2"
   - View charts and tables
   - Export to PDF/Excel if needed

